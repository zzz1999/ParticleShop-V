# ParticleShop 
A magical particle trail plugin:D<br>
一个神奇的粒子跟踪插件。:D
# Usages
- /ps set <number><br>
- 设置粒子 | choose the kind of particles<br>
- /ps stop<br>
- 关闭粒子 | Prevent the partcles<br>
- /ps list<br>
- 列出你所能使用的粒子列表 | list the list of particles that you can use
